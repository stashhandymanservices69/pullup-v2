# 03 — Accounting Setup MVP (Australia)

Not tax advice. This is an execution checklist to start clean from day one.

## Goal (1 Hour Setup)
Get accounting ready before launch so GST, Stripe fees, refunds, and commission income are recorded correctly from the first transaction.

## 1) Tooling and Connections
- [ ] Choose accounting platform (Xero / MYOB / QuickBooks AU)
- [ ] Connect business bank account feed
- [ ] Connect Stripe feed/integration
- [ ] Confirm business details (ABN, legal entity name, address)

## 2) Chart of Accounts (Minimum)
- [ ] Sales / Platform income
- [ ] Commission revenue
- [ ] Stripe processing fees
- [ ] Refunds issued
- [ ] Chargebacks/disputes
- [ ] GST collected (liability)
- [ ] GST paid/credits (if applicable)

## 3) Tax Configuration
- [ ] GST settings enabled and default tax codes reviewed
- [ ] Correct tax treatment for commissions configured
- [ ] Correct tax treatment for refunds configured
- [ ] Test transaction posted and verified with expected GST outcome

## 4) Documents and Templates
- [ ] Invoice template ready
- [ ] Receipt template ready
- [ ] Credit note/refund template ready
- [ ] Naming conventions set for clean audit trail

## 5) Reconciliation Process (Weekly Minimum)
- [ ] Match orders to Stripe payments
- [ ] Match Stripe payouts to bank deposits
- [ ] Verify Stripe fees captured
- [ ] Verify refunds/chargebacks captured
- [ ] Close week with signed checklist owner/date

## 6) Accountant Validation (One-Time Early)
- [ ] Ask accountant to review account mappings
- [ ] Ask accountant to review GST settings for your model
- [ ] Lock rules after approval to avoid drift

## Done Definition
Accounting MVP is complete when:
- [ ] First live transaction can be fully traced from order → Stripe → bank → books
- [ ] GST outcome is predictable and validated
- [ ] Weekly reconciliation owner is assigned
